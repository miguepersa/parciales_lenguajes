## Calculadora de expresiones matematicas

### Ejecucion
```python calculadora.py```

### Comandos
```
MOSTRAR <tipo> <expresion>
```

```
EVAL <tipo> <expresion>
```

```
SALIR
```

### Tests
Se requiere tener instalado pytest en el sistema. Para instalarlo, ejecutar:
`pip install pytest`

Para ejecutar los tests:
```
pytest
```
o
```
python -m pytest
```
